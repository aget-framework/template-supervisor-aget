# Prerequisites

## System Requirements
- Python 3.8+
- Git 2.0+
- Bash shell (zsh/bash)
- 100MB free disk space

## Python Dependencies
All scripts use standard library only - no pip install required.

## Quick Verification
```bash
# Check Python
python3 --version  # Should be 3.8+

# Check Git
git --version      # Should be 2.0+

# Check shell
echo $SHELL        # Should show bash or zsh
```

## Full Validation
Run the validation script after setup:
```bash
./validate.sh      # Will be added in Gate 3
```

## Platform Notes
- **macOS**: All requirements typically pre-installed
- **Linux**: May need to install python3
- **Windows**: Use WSL2 or Git Bash
