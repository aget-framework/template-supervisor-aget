# Patterns Directory

This directory contains discovered and validated patterns from AGET usage.

## Naming Convention
All pattern files follow the `UPPER_CASE_PATTERN.md` convention:
- ✅ `AGENT_MISSION_VISION_PATTERN.md`
- ✅ `HIGH_RISK_FRICTION_PATTERN.md`
- ❌ `agent-mission-pattern.md` (wrong)
- ❌ `north_star_pattern.md` (wrong)

## Pattern Categories
- `governance/` - Strategic and organizational patterns
- `identity/` - Agent identity and personality patterns
- `naming/` - Naming conventions and safety patterns
- `security/` - Security and key management patterns
- `safety/` - Risk mitigation and friction patterns
- `creativity/` - Ideation and capture patterns
- `migration/` - System migration patterns

## Pattern Status Levels
- **DISCOVERED** - Seen in practice, not yet standardized
- **VALIDATED** - Tested and confirmed useful
- **GRADUATED** - Promoted to public AGET framework
